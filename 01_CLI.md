## CLI

Command Line Interface (명령형 인터페이스0



## 명령어들

- `pwd`  현재 내가 있는 위치 (print working directory)
- `ls` : 현재 위치의 파일 및 폴더를 출력(list) 
- `cd  ` 폴더를 변경 (change directory)
- `..`상위 폴더로 이동.
- `mkdir`[폴더명]: 새로운 폴더를 생성(make directory)
- `rm[파일명]`  파일을 삭제(remove)
- `rm -r [폴더명]` 폴더를 삭제

- `ls -a` 숨긴 파일 보기
